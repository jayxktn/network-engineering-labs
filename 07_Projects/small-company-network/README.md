# Small Company Network (VLAN + Inter-VLAN Routing)

## Overview
This project simulates a small enterprise network using Cisco Packet Tracer. It demonstrates VLAN segmentation, trunking, and router-on-a-stick inter-VLAN routing.

## Objectives
- Segment network into departments using VLANs
- Enable communication between VLANs using a Layer 3 router
- Implement 802.1Q trunking between switch and router
- Validate connectivity using ICMP testing (ping)

## Network Design
- VLAN 10: ADMIN
- VLAN 20: SALES
- VLAN 30: IT

## Key Technologies
- VLANs (Layer 2 segmentation)
- 802.1Q Trunking
- Inter-VLAN Routing (Router-on-a-Stick)
- IPv4 Subnetting

## Results
- Successful inter-VLAN communication
- Stable routing between all departments
- Verified end-to-end connectivity

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI
- Basic network troubleshooting (ping, show commands)
